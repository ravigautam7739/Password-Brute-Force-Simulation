# 💻 Password Brute Force Simulation

This Python project demonstrates how a **brute force password attack works**.

The program attempts multiple passwords from a predefined list until it finds the correct one.

⚠️ This project is created **only for educational purposes** to help understand how brute-force attacks work.

---

# 🚀 Features

✔ Simulates a brute force password attack
✔ Displays each password attempt
✔ Stops when the correct password is found
✔ Beginner-friendly cybersecurity demonstration

# 🧠 How It Works

1. A list of common passwords is created.
2. The program tries each password one by one.
3. When the correct password is found, the system grants access.

This demonstrates the concept of **brute force attacks used in cybersecurity**.
